# TAU course on WebDriverIO

[Instructor: Julia Pottinger](https://testautomationu.applitools.com/webdriverio-tutorial/)

## Setup

Under `cd tau-webdriverio`

```zsh
npm install
```

## Running tests

All tests

```zsh
npm run test
```

A specific spec

```zsh
npm run test -- --spec ./test/specs/hover.test.js
```
